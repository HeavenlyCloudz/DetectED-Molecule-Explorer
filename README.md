# 🧬 DetectED Molecule Explorer

An interactive cheminformatics teaching app built with **RDKit** and **Streamlit** for **DetectED Labs**, an extension of DetectED. Our goal is to make early disease detection and AI education accessible, this time through hands-on molecular exploration!

---

## ✨ Features

- **SMILES Input** – Type any SMILES string or pick from curated examples (Aspirin, Ibuprofen, Caffeine, etc.)
- **Auto‑Name Lookup** – Fetches the common drug name from PubChem (when available)
- **2D Structure Viewer** – Displays high‑quality 2D molecular images with download option
- **3D Interactive Viewer** – Rotate, zoom, and explore molecules in 3D (optional)
- **Molecular Properties** – Computes key descriptors:
  - Molecular Weight, LogP, TPSA
  - H‑bond Donors/Acceptors, Rotatable Bonds
  - Fraction Csp³, Aromatic/Saturated/Total Rings, Heavy Atoms
- **Lipinski Rule of 5** – Automatic pass/fail check with detailed violation breakdown
- **Drug‑Likeness Summary** – Plain‑language interpretation of bioavailability potential
- **Random Molecule** – Surprise button for exploration and classroom engagement
- **Viewing History** – Tracks recently viewed molecules for easy revisiting
- **Fun Facts** – Historical and biological context for common drugs
- **Quiz Mode** – Gamified button to test students' knowledge
- **Downloadable Images** – Save 2D structures as PNG files

---

## ▶️ How to Use
- Choose an input method: Select from examples, Type SMILES, or Random molecule
- View the 2D structure and automatically fetched name
- Explore molecular properties and Lipinski compliance
- Toggle the 3D viewer (if installed) for interactive exploration
- Download the image or test yourself with the "Quiz Me!" button

## 📚 Teaching Applications
This app is designed for classroom and workshop settings to teach:
- Cheminformatics basics – SMILES notation, molecular representations
- Drug discovery principles – Lipinski's Rule of 5 and drug‑likeness
- Computational chemistry – Descriptors and property prediction
- Critical thinking – Interpreting property data for drug design
